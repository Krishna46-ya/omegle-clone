# Omegle Clone with WebRTC

A real-time anonymous video chat application that connects users with random strangers, similar to Omegle.  
built using **webrtc** for peer-to-peer video communication.

---

## features

- random stranger matching
- real-time video chat using webrtc
- peer-to-peer communication
- anonymous connections
- separate frontend and backend

---

## requirements

make sure you have the following installed:

- node.js / npm
- git
- typescript (`tsc`)

---

## development setup

clone the repository and install dependencies for both frontend and backend.

```
git clone https://github.com/krishna46-ya/omegle-clone
cd omegle-clone

cd backendmeet
npm install

cd ../frontendgmeet
npm install

cd ..
```

## running the project
you need **two terminal windows** to run the frontend and backend servers.

##### start backend
```
cd cd backendmeet
npm run dev
```
##### start Frontend 
```
cd frontendGMeet
npm run dev
```
